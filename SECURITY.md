## Reporting a Vulnerability

If you discover a security vulnerability within Hookx, please send an e-mail to Ali Harb via [harbzali@gmail.com](mailto:harbzali@gmail.com). All security vulnerabilities will be promptly addressed.

Please do **not** report security vulnerabilities through public GitHub issues.
